# Giant Header

Lightweight Gutenberg blocks for site header and navigation (52KB total).

## Features
- Site Header block with logo upload, search bar, mobile menu
- Navigation Menu block with nested submenus
- Three menu types: Simple Link, Parent with URL, Parent without URL
- Native Gutenberg fields (no ACF)
- Fully responsive

## Installation
1. Upload `giant-header` folder to `/wp-content/plugins/`
2. Activate in WordPress admin
3. Add "Site Header" block to template
4. Configure in block sidebar

## Usage
**Header Block:** Upload logo in sidebar → Navigation menu auto-included
**Navigation Menu:** Add items in sidebar → Choose type → Add submenus & children

Total size: 52KB (no node_modules!)
